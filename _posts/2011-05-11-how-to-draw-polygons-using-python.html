In this tutorial I will show you how easy it is to draw polygons using Python. We will do this using the Pygame library. At the end of the article I will present a complete program that draws some polygons.

I assume you have at least a basic understanding of Python. If you have never used the Pygame library, please consider reading the tutorial A Fast Introduction to Pygame.

So, let's go...

To draw polygons we use the polygon() function from the Draw module. See the signature of the function below.

pygame.draw.polygon(Surface, color, pointlist, width=0)

The first parameter specifies the surface where the polygon will be drawn into. The color parameter specifies the color that will be used to fill the polygon. The pointlist parameter specifies the list of vertices of the polygon. The polygon vertices are represented as (X,Y) tuples. If width is zero the polygon is filled, otherwise only its edges are drawn.

Let us see some examples of the usage of the function below:

 

[python]
import pygame, sys

RED = (255,0,0)
GREEN = (0,255,0)

pygame.init()
screen = pygame.display.set_mode((640,480))

while True:
for event in pygame.event.get():
if event.type == pygame.QUIT:
pygame.quit()
sys.exit()

# Clear the window surface
screen.fill((0,0,0))

# Draw a red triangle
pygame.draw.polygon(screen, RED, [(100,10),(10,50),(190,50)])

# Draw a rectangle
pygame.draw.polygon(screen,GREEN,[(200,40),(300,40),(300,100),(200,100)])

# Display the polygons in the window.
pygame.display.flip()
